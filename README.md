## 🏛️ O Conceito
O **Paper-Contracts** transcende a categoria de "geradores de documentos". É um **Motor Editorial de Precisão** desenvolvido para profissionais que exigem que a estética de seus contratos, faturas e currículos projete autoridade instantânea. Inspirado na tipografia suíça e no minimalismo clássico, o sistema elimina o ruído visual e o aspecto "gerado por IA" para entregar resultados de nível editorial.

## ✨ Diferenciais da Versão Final (Enterprise Edition)

### 1. 🎨 Estilo do Papel (Modular Design)
Diferente de templates rígidos, o Paper-Contracts oferece **Soberania Editorial**. Através do painel de estilo, o usuário controla dinamicamente:
- **Identidade Modular**: Ative ou desative Cabeçalhos, Rodapés e Selos de Status de forma independente.
- **Marca d'Água Forense**: Aplique assinaturas digitais visuais (Biometria/Fingerprint) que reforçam a autenticidade.
- **Tipografia Adaptativa**: Alternância entre fontes Serifadas (Luxo/Tradicional) e Sans-Serif (Moderno/Tech).

### 2. 🧠 ATS & Risk Intelligence
- **ATS Analyzer**: Motor semântico que valida currículos contra algoritmos de triagem corporativos.
- **Risk Analysis**: IA treinada para identificar lacunas em cláusulas e sugerir mitigações em tempo real.

### 3. 🔐 Segurança de Dados & Privacidade
- **Firebase Auth Integration**: Autenticação segura via Google, GitHub ou E-mail com proteção de rotas via `firebase-admin`.
- **Zero-Knowledge Architecture**: Processamento focado em privacidade, garantindo que o conteúdo do documento pertença apenas ao usuário.

## 🛠️ Stack Tecnológica

- **Frontend:** React 19 + Vite (SPA de Alta Performance).
- **Backend:** Node.js (Express) + Firebase Admin SDK.
- **IA:** Gemini 1.5 Pro (via Google Vertex AI SDK).
- **Auth/Hosting:** Firebase Cloud Functions & Vercel.
- **Estilização:** CSS Vanilla de Alta Precisão (Sem frameworks invasivos).

## 🚀 Como Rodar o Projeto

1. **Instalação**:
   ```bash
   npm install
   ```

2. **Variáveis de Ambiente**:
   Configure o `.env` no backend com sua `GEMINI_API_KEY` e as credenciais do Google Cloud.

3. **Execução**:
   ```bash
   npm run dev
   ```
